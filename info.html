By combining all the table using sql as such:

  • The average score of each exhibition according to reviews
  • with cte as (select *,
  • case when ReviewQualityFK = 1 then 1
  • when ReviewQualityFK = 2 then 0
  • else 0.5 end as new_score
  • from ReviewandArtist ra
  • inner join Artists a
  • on ra.ArtistFK = a.ArtistsPK)
  • select ExhibitionFK, avg(new_score) as average_score from cte
  • group by ExhibitionFK
